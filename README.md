## Introduction

This is an express server for a react app, to be used for planning.

https://react-kanban-board-main.netlify.app/

## Setup

- Fork this repo

- Clone this repo

- Add variables to ".env" file in root folder:
```shell
PORT=
MONGODB_URI=
TOKEN_SECRET=
SALT_ROUNDS=
```

- run in shell/cmd:
```shell
cd react-kanban-board
npm install
npm run dev
```
